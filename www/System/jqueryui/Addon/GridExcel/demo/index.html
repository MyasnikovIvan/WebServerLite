A jQuery plugin that transforms JSON data into an Excel-style data grid with advanced features like filtering, sorting, editing, progressive data loading, and much more.

Features:

  • ✅ Cells navigation like Excel
  • ✅ Progressive data loading
  • ✅ Search
  • ✅ Columns sorting
  • ✅ Refresh table
  • ✅ Update cell (text, dropdown, date, time, comment)
  • ✅ Delete rows
  • ✅ Excel export (need TableToExcel.js)
